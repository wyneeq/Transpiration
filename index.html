Help - Shortcuts

control + A Lowers the opacity / Opaklığı düşürür

control + D Increases opacity / Opaklığı arttırır

toggle control + T Sets Opacity to opposite Opacity / Opaklığı zıt Opaklığa ayarlar

toggle control + L Pin on top / En üstte tuttur

control + W Closes the program / Programı kapatır

control + Q Upload new image / Yeni resim yükle

control + C Centers the program / Programı merkezler

toggle control + H Shows the help menu / Yardım menüsünü gösterir
